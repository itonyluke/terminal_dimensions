### this program outputs terminal dimensions, rows by columns and pixels by pixels

#### install:
`git clone https://github.com/itonyluke/terminal_dimensions terminal_dimensions`

#### compile:
`gcc terminal_dimensions.c`

#### run:
`./a.out`
